## 📌 Project Overview

This project focuses on predicting the price of cars using Machine Learning techniques.  
The dataset contains information about various car attributes such as company, model, fuel type, and other specifications.

The objective is to build a regression model that can estimate the price of a car based on its features.

---

## 🎯 Problem Statement

Car prices vary based on many factors such as brand, fuel type, engine capacity, and year of manufacture.

This project aims to develop a **machine learning model that predicts car prices** using historical car data.

---

## 📊 Dataset Information

The dataset includes information such as:

| Feature | Description |
|------|------|
| Car Name | Name of the car |
| Company | Car manufacturer |
| Year | Manufacturing year |
| Fuel Type | Petrol / Diesel / CNG |
| Kilometers Driven | Distance driven |
| Price | Target variable |
